# Microsoft Defender XDR

Microsoft Defender XDR provides extended detection and response across identities, endpoints, email, applications, and cloud assets.

---

## Services
- **Endpoints** with Microsoft Defender for Endpoint
- **Assets** with Microsoft Defender Vulnerability Management
- **Email and collaboration** with Microsoft Defender for Office 365
- **Identities** with Microsoft Defender for Identity
- **Applications** with Microsoft Defender for Cloud Apps

---

## Microsoft Defender for Office 365
- Preset security policies
- Threat protection policies
- Reports
- Threat investigation and response capabilities
- Automated investigation and response capabilities

Safeguards against malicious threats from:
- Email messages
- URLs (links)
- Microsoft Teams
- SharePoint Online
- OneDrive for Business
- Other Office clients

---

## Microsoft Defender for Endpoint
- Core Defender Vulnerability Management
- Attack Surface Reduction (ASR)
- Next-generation protection
- Endpoint Detection and Response (EDR)
- Automated Investigation and Remediation (AIR)
- Microsoft Secure Score for Devices
- Microsoft Threat Experts
- Management and APIs

Designed to prevent, detect, investigate, and respond to advanced endpoint threats.

---

## Microsoft Defender for Cloud Apps
A Cloud Access Security Broker (CASB) providing:
- Fundamental CASB functionality
- SaaS Security Posture Management (SSPM)
- Advanced threat protection
- App-to-app protection

Supports deployment modes:
- Log collection
- API connectors
- Reverse proxy

CASB pillars:
- Visibility
- Compliance
- Data security
- Threat protection

---

## Microsoft Defender for Identity
Cloud-based identity security solution detecting advanced threats and malicious activities.

### Cyberattack Kill Chain
- **Reconnaissance**: Detect rogue users and information-gathering attempts
- **Compromised credentials**: Identify brute force attacks, failed authentications, and suspicious changes
- **Lateral movements**: Detect attempts to move within the network
- **Domain dominance**: Detect attacker control over domain controllers

---

## Microsoft Defender Vulnerability Management

### Visibility and Assessment
- Software and vulnerability visibility
- Network share assessment
- Browser extensions assessment
- Digital certificates assessment

### Remediation and Tracking
- Remediation requests sent to IT (via Microsoft Intune)
- Block vulnerable applications
- Alternate mitigations (configuration-based risk reduction)
- Real-time remediation status tracking

---

## Microsoft Defender Threat Intelligence
- **Threat Analytics**: Expert-driven intelligence highlighting relevant emerging threats
- **Intel Profiles**: Curated intelligence by threat actors, tools, and vulnerabilities
- **Intel Explorer**: Central access to Defender Threat Intelligence content

- Compliance covers data residency and regulatory requirements (e.g., GDPR, PCI).

### Feature Clarifications
- **Attack Surface Reduction (ASR)**: Handles access to malicious endpoints
- **Automated Investigation and Remediation (AIR)**: Uses playbooks to analyze alerts and take action
- **Microsoft Threat Experts**: Microsoft-managed SOC expertise
- **Threat & Vulnerability Management**: Scans for vulnerabilities and misconfigurations
