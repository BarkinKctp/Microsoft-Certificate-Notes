# Capabilities of Power Pages

## Power Pages
Power Pages helps organizations:
- Improve customer interactions  
- Simplify data operations  
- Improve day-to-day operations  
- Build relationships with buyers  

---

## Build a Power Pages Site
Steps to build a site:
- Create a website  
- Define the pages  
- Select a theme and brand your site  
- Connect to Dataverse data  
- Secure your site  

---

## Design Studio
The primary tool for building Power Pages sites is **Design Studio**.  
It simplifies:
- Creating and modifying site pages  
- Connecting sites to Microsoft Dataverse  
- Defining site security  

Design Studio is organized into workspaces, each focused on a specific function.

### Design Studio Workspaces
- **Pages workspace**: Create, edit, and arrange webpages.
- **Styling workspace**: Customize the look and feel with themes and branding.
- **Data workspace**: Create and modify Microsoft Dataverse tables.
- **Set up workspace**: Administrative area for site management and configuration.

---

## Power Pages Templates
When creating a Power Pages website, a template must be selected.

Template types:
- **Scenario-based templates**: Complete sites for common scenarios such as registrations, permit applications, and meeting bookings.
- **Site design templates**: Prebuilt sites with pages, layouts, images, and colors.
- **Blank page template**: A single home page where you add all other content manually.
- **Dynamics 365 templates**: Require the related Dynamics 365 app to be installed in the environment.

---

## Components in Power Pages

### Data Components
- **Form component**: Displays rows from a Dataverse table and allows creating and editing rows.
- **List component**: Displays a read-only view of Dataverse table data.

### Other Components
- **Text component**: Displays static text only.
- **iframe component**: Embeds external web pages using a URL and cannot display Dataverse data.

---

## Dataverse and Integrations
- Forms allow external users to create, view, and edit Dataverse table rows.
- Power BI content can be embedded in Power Pages sites, but visualizations are created in Power BI.
- System integrations are handled using Power Automate cloud flows and connectors.
- AI-powered chat experiences are built using Copilot Studio.
